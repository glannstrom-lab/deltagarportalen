## 1. Nuvarande läge

### ✅ Vad vi har åstadkommit
- **Kunskapsbanken** utökad med 30+ artiklar kopplade till övningar
- **Övningar** fungerar med molnlagring av svar
- **CV-generator** och intresseguide på plats
- **AI-assistent** integrerad för stöd
- **Jobbsökning** via Arbetsförmedlingens API
- **Autentisering** via Supabase Auth

### ⚠️ Identifierade problem (senaste veckan)
1. **Databas-konfiguration:** RLS-policyer saknas för flera tabeller (article_reading_progress, etc.)
2. **Datakonsistens:** Tags hanteras ibland som array, ibland som string – skapar runtime-fel
3. **Felhantering:** Supabase-fel (400, 403, 42501) visas i konsolen och stör användarupplevelsen
4. **Offline-stöd:** Begränsad – appen kräver konstant uppkoppling
5. **Prestanda:** Laddningstider för artiklar och övningar kan optimeras

- [ ] **Åtgärd:** Lägg till databasmigreringar för konsistens

```sql
-- Exempel på RLS-policy som saknas:
CREATE POLICY "Users can update own reading progress"
ON article_reading_progress
FOR ALL
TO authenticated
USING (user_id = auth.uid())
WITH CHECK (user_id = auth.uid());
```
