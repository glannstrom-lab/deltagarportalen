VAD DU BEHÖVER
--------------
1. En server/host med antingen:
   - Node.js-stöd (VPS, Render, Railway, etc.) ELLER
   - PHP-stöd (Loopia, One.com, de flesta webbhotell)

2. En OpenRouter API-nyckel:
   - Gå till https://openrouter.ai
   - Skapa konto (gratis att starta)
   - Skapa API-nyckel
   - Kopiera nyckeln (börjar med "sk-or-v1-")

INSTALLATION - NODE.JS
----------------------
1. Packa upp nodejs-version.zip
2. Redigera .env.example → spara som .env
3. Fyll i din OPENROUTER_API_KEY
4. Ladda upp till din server
5. Kör: npm install && npm start

INSTALLATION - PHP
------------------
1. Packa upp php-version.zip
2. Redigera config.php (fyll i API-nyckel)
3. Ladda upp till din webbhotells public_html/
4. Gå till din-domän.com/public/

SÄKERHET
--------
⚠️  Lägg ALDRIG din API-nyckel i frontend-kod
⚠️  Lägg ALDRIG .env eller config.php i git
✅ API-nyckeln är säker på servern i båda versionerna
