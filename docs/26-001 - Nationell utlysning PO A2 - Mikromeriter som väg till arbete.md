# EU-projekt: Mikromeriter som väg till arbete

**Diarienummer:** 26-001
**Utlysning:** Nationell utlysning PO A2
**Utlyst belopp:** 80 000 000 kr
**Medfinansiering:** 46%
**Ansökningsperiod:** 2026-02-12 – 2026-06-04
**Projektperiod:** 2026-10-01 – 2029-06-30

---

## Sammanfattning av EU-utlysningen

### Syfte
Genom stärkta och synliggjorda mikromeriter och generella kompetenser, förbättra anställningsbarheten hos arbetslösa som bedöms stå långt ifrån arbetsmarknaden.

### Målgrupp
- Arbetslösa personer som bedöms stå långt ifrån arbetsmarknaden
- Personer som saknar formell utbildning eller meriter som erkänns av arbetsgivare
- Särskilt: låg utbildningsnivå, funktionsnedsättning, hög ålder, utrikes födda
- Långtidsarbetslösa

### Vad är mikromeriter?
Mikromeriter är dokumentation av läranderesultat från mindre, avgränsade lärandeinsatser. De kan visa att en individ har utvecklat specifika kunskaper, färdigheter eller kompetenser och kan förvärvas genom:
- Utbildning
- Arbetslivserfarenhet
- Andra former av lärande (informellt/icke-formellt)

Mikromeriter kan synliggöra både:
- **Yrkesspecifika kompetenser:** certifikat, licenser, bevis
- **Generella kompetenser:** samarbetsförmåga, problemlösning, digital grundkompetens, arbetslivskunskap, flexibilitet

### Inriktning på insatser
1. Bedömning av mikromeriter
2. Stöd i att få mikromeriter dokumenterade
3. Stöd i att synliggöra mikromeriter
4. Kompetensutveckling för att uppnå nivå för att få mikromeriter dokumenterade
5. Stöd i efterföljande matchningsprocess i strävan att finna arbete

### Förväntade resultat
**Krav:**
- Deltagare har fått kompetenser bedömda i form av mikromeriter
- Kompetenser dokumenterade
- Verktyg att synliggöra dessa

**Önskvärt:**
- Nya metoder för bedömning och synliggörande prövade
- Kompetensglapp identifierade och åtgärdade
- Nya samarbetsformer utvecklade
- Plan för tillvaratagande av resultat efter projektet

## Detaljerade funktionsbeskrivningar

### 1. Mikromeritvärderare (AI-driven kompetensbedömning)

**Beskrivning:**
En AI-driven modul där deltagaren svarar på strukturerade frågor om sin bakgrund, livserfarenhet, volontärarbete, hobbyer och tidigare arbete. AI:n identifierar och kategoriserar dolda kompetenser enligt SeQF nivå 1-4.

**Funktionalitet:**
- Guidad intervju om bakgrund och erfarenheter
- AI-analys av svar för att identifiera kompetenser
- Förslag på mikromeriter baserat på erfarenheter
- Mappning till SeQF-nivåer (1-4, upp till gymnasienivå)
- Möjlighet för konsulent att verifiera bedömningen

**Koppling till EU-krav:** *"Bedömning av mikromeriter"*

---

### 2. Digital Kompetensportfölj

**Beskrivning:**
En samlingsplats där alla bedömda och dokumenterade mikromeriter lagras i ett standardiserat format (inspirerat av EU:s Europass). Kan exporteras som PDF eller delbar länk till arbetsgivare.

**Funktionalitet:**
- Översikt över alla mikromeriter
- Kategorisering (yrkesspecifika vs generella)
- Verifieringsstatus (självskattad/verifierad av konsulent)
- SeQF-nivå för varje mikromerit
- Export till PDF
- Delbar länk för arbetsgivare
- Integration med LinkedIn

**Koppling till EU-krav:** *"Stöd i att få mikromeriter dokumenterade"* och *"verktyg att synliggöra dessa"*

---

### 3. Kompetensbevis-generator

**Beskrivning:**
Skapa visuella digitala badges/certifikat för varje dokumenterad mikromerit. Följer Open Badges-standard och kan delas på LinkedIn, CV, eller via QR-kod.

**Funktionalitet:**
- Generera visuella badges för mikromeriter
- QR-kod för verifiering
- Delning till sociala medier
- Nedladdning som bild/PDF
- Verifieringskedja (vem har bedömt/verifierat)

**Koppling till EU-krav:** *"Stöd i att synliggöra mikromeriter"*

---

### 4. Informellt Lärande-logg

**Beskrivning:**
Dagboksliknande verktyg där deltagare kontinuerligt dokumenterar lärande från vardagen - volontärarbete, föräldraskap, hobby-projekt, praktik. AI:n föreslår vilka kompetenser som demonstreras.

**Funktionalitet:**
- Enkel loggning av dagligt lärande
- Kategorisering av aktivitetstyp
- AI-förslag på relaterade kompetenser
- Koppling till potentiella mikromeriter
- Tidslinje över läranderesa
- Export för sammanställning

**Koppling till EU-krav:** *"Mikromeriter kan förvärvas genom... andra former av lärande"*

---

### 5. Generella Kompetenser-självvärdering

**Beskrivning:**
Strukturerad bedömning av mjuka färdigheter som specifikt nämns i utlysningen: samarbetsförmåga, problemlösning, flexibilitet, digital grundkompetens, kommunikation, arbetslivsfärdigheter.

**Funktionalitet:**
- Självskattning på skala
- Scenariobaserade frågor för validering
- Jämförelse mot arbetsmarknadens krav
- Generering av mikromeriter för bekräftade kompetenser
- Tips för förbättring

**Kompetenser som bedöms:**
- Samarbetsförmåga
- Problemlösning
- Flexibilitet
- Digital grundkompetens
- Kommunikation
- Arbetslivsfärdigheter
- Initiativförmåga
- Stresshantering

**Koppling till EU-krav:** *"generella kompetenser som exempelvis samarbetsförmåga, problemlösningsförmåga, flexibilitet, digital kompetens"*

---

### 6. Digital Grundkompetens-certifiering

**Beskrivning:**
Interaktiva lärmoduler för grundläggande digital kompetens (baserat på DigComp-ramverket) med avslutande bedömning som genererar en verifierbar mikromerit.

**Moduler:**
1. Grundläggande datorkunskap
2. Internet och säkerhet
3. E-post och digital kommunikation
4. Ordbehandling (grundläggande)
5. Kalkylprogram (grundläggande)
6. Jobbsökning online
7. Digitala möten och samarbetsverktyg

**Funktionalitet:**
- Interaktiva lektioner
- Praktiska övningar
- Avslutande quiz/bedömning
- Certifikat vid godkänt resultat
- Spårning av framsteg

**Koppling till EU-krav:** *"Kompetensutveckling för att uppnå nivå för att få mikromeriter dokumenterade"*

---

### 7. Bransch-Mikromeritkarta

**Beskrivning:**
Visar vilka mikromeriter som efterfrågas inom olika branscher/yrken. Matchar deltagarens befintliga kompetenser mot branschkrav och visar glapp att fylla.

**Funktionalitet:**
- Välj bransch eller yrke
- Se efterfrågade mikromeriter
- Jämför med egna kompetenser
- Identifiera glapp
- Föreslå vägar att fylla glappen
- Koppling till kompetensutvecklingsmoduler

**Branscher:**
- Vård och omsorg
- Handel och service
- Bygg och anläggning
- Transport och logistik
- Hotell och restaurang
- Administration och kontor
- IT och teknik
- Industri och produktion

**Koppling till EU-krav:** *"kvalitetssäkrade branschmodeller"* och *identifiering av kompetensglapp*

---

### 8. Stegförflyttnings-tracker

**Beskrivning:**
Visualiserar deltagarens resa från nuläge mot arbete med konkreta delmål. Varje uppnådd mikromerit = ett steg framåt. Gamification med milstolpar och uppmuntran.

**Funktionalitet:**
- Visuell tidslinje/resa
- Milstolpar kopplade till mikromeriter
- Progress-indikatorer
- Firande vid uppnådda mål
- Historik över framsteg
- Delning av framsteg med konsulent

**Koppling till EU-krav:** *"stegförflyttningar mot att bryta sin arbetslöshet"*

---

### 9. Arbetsgivarmatchning baserat på Mikromeriter

**Beskrivning:**
Intelligent matchning som kopplar deltagarens dokumenterade mikromeriter mot platsannonser. Visar matchningsgrad och föreslår vilka ytterligare mikromeriter som skulle öka chanserna.

**Funktionalitet:**
- Matchningsindikator på jobbannons (t.ex. "Du matchar 73%")
- Filter: "Visa jobb baserat på mina mikromeriter"
- Identifiering av saknade kompetenser
- Direktlänk till relevant kompetensutveckling
- Prioritering av jobb baserat på matchning

**Koppling till EU-krav:** *"Stöd i efterföljande matchningsprocess i strävan att finna arbete"*

---

### 10. Konsulent-dashboard för Mikromeritracking

**Beskrivning:**
Vy för arbetskonsulenter/handledare där de kan följa deltagarens mikromerit-resa, verifiera kompetenser och exportera rapporter.

**Funktionalitet:**
- Översikt över deltagarens mikromeriter
- Framsteg i pågående certifieringar
- Verifiera/attestera bedömda kompetenser
- Kommentarfunktion
- Export av rapporter (för Arbetsförmedlingen)
- Aggregerad statistik över alla deltagare

**Koppling till EU-krav:** *Krav på rapportering till SCB och uppföljning*

### Databasschema (Supabase)

**Nya tabeller:**

```sql
-- Mikromeriter
CREATE TABLE micro_credentials (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES auth.users(id),
  title TEXT NOT NULL,
  description TEXT,
  category TEXT, -- 'vocational' | 'general'
  seqf_level INTEGER CHECK (seqf_level BETWEEN 1 AND 4),
  status TEXT DEFAULT 'self_assessed', -- 'self_assessed' | 'verified' | 'certified'
  verified_by UUID REFERENCES auth.users(id),
  verified_at TIMESTAMP,
  evidence JSONB, -- länkar till bevis
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

-- Lärandelog
CREATE TABLE learning_log (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES auth.users(id),
  date DATE NOT NULL,
  activity TEXT NOT NULL,
  description TEXT,
  competencies TEXT[], -- relaterade kompetenser
  micro_credential_id UUID REFERENCES micro_credentials(id),
  created_at TIMESTAMP DEFAULT NOW()
);

-- Certifieringsframsteg
CREATE TABLE certification_progress (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES auth.users(id),
  certification_type TEXT NOT NULL,
  module_id TEXT NOT NULL,
  status TEXT DEFAULT 'not_started', -- 'not_started' | 'in_progress' | 'completed'
  score INTEGER,
  completed_at TIMESTAMP,
  created_at TIMESTAMP DEFAULT NOW()
);

-- Kompetensbevis
CREATE TABLE competency_badges (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  micro_credential_id UUID REFERENCES micro_credentials(id),
  badge_image_url TEXT,
  verification_code TEXT UNIQUE,
  issued_at TIMESTAMP DEFAULT NOW(),
  expires_at TIMESTAMP
);
```
