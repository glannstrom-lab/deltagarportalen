# EU-projekt: Stärkt kompetens för ett arbetsliv med AI

**Diarienummer:** 26-002
**Utlysning:** Nationell utlysning POA1
**Utlyst belopp:** 45 000 000 kr
**Medfinansiering:** 46%
**Ansökningsperiod:** 2026-02-24 – 2026-06-04
**Projektperiod:** 2026-10-01 – 2029-06-30

---

## Sammanfattning av EU-utlysningen

### Syfte
Organisationer och deras anställda ska få kompetensutveckling för att stå bättre rustade för ett arbetsliv med AI.

### Målgrupp
- Anställda kvinnor och män som behöver kompetensutveckling kopplat till AI
- Medarbetare och ledare, oavsett anställningsform
- **Personer som riskerar arbetslöshet**
- Arbetskonsulenter och handledare som stöttar jobbsökande

### Bakgrund
AI Swedens insiktsrapport (2025) konstaterar att utvecklingen av AI skapar:
- Betydande möjligheter: ökad produktivitet, innovation, stärkt välfärd
- Omfattande omställningsrisker: kompetensglapp, obalanser i styrning

### Lärdomar från förstudier
Kompetensutvecklingsprojekt inriktade mot AI bör:
1. **Styra mot organisatorisk och kulturell förändringsprocess** – inte enbart teknisk
2. **Bygga grundläggande AI-förståelse** som bas
3. **Stärka mänskligt omdöme, etik och ansvar** i AI-användning
4. **Ta höjd för förändrade roller** och arbetsprocesser
5. **Beakta ledarskapets roll** i AI-omställningen
6. **Bygga på verksamhetsnära lärande** – kontinuerligt och praktiskt

### Förväntade resultat
- Deltagande organisationer har stärkt sin förmåga att förstå, använda och förhålla sig till AI
- Anställda har utvecklat relevant kompetens för ett arbetsliv med AI
- Kompetensutvecklingsinsatser har stärkt deltagarnas ställning på arbetsmarknaden

## Detaljerade funktionsbeskrivningar

### 1. AI-Kompetensguide (Grundläggande AI-förståelse)

**Beskrivning:**
Interaktiv introduktion till AI för jobbsökande. Förklarar vad AI är, hur det fungerar, och varför det är viktigt på arbetsmarknaden. Ingen förkunskap krävs.

**Funktionalitet:**
- Modulbaserad inlärning (5-10 min per modul)
- Vad är AI? (definitioner, typer av AI)
- Hur används AI idag? (exempel från arbetslivet)
- AI vs mänsklig intelligens (styrkor och begränsningar)
- AI i din bransch (anpassat till användarens intresse)
- Quiz och kunskapskontroll efter varje modul
- Certifikat vid avslutad kurs

**Moduler:**
1. Introduktion till AI
2. Machine Learning förklarad
3. Generativ AI och språkmodeller
4. AI på arbetsplatsen
5. Din roll i en AI-driven arbetsmarknad

**Koppling till EU-krav:** *"Bygga upp grundläggande AI-förståelse som bas för kompetensutveckling"*

---

### 2. AI-verktyg för Jobbsökning

**Beskrivning:**
Praktisk guide för att använda AI-verktyg i jobbsökningsprocessen. Lär deltagarna att effektivt använda portalens AI-funktioner samt externa verktyg som ChatGPT, Copilot, och branschspecifika AI-verktyg.

**Funktionalitet:**
- Steg-för-steg-guider för portalens AI-funktioner
- Tutorial: Använd AI för CV-förbättring
- Tutorial: AI-genererade personliga brev
- Tutorial: Intervjuförberedelse med AI
- Tutorial: LinkedIn-optimering med AI
- Praktiska övningar med feedback
- Tips om externa AI-verktyg för jobbsökning

**Praktiska övningar:**
- Skapa ett CV med AI-stöd
- Anpassa ett personligt brev till en specifik annons
- Öva intervjusvar med AI-feedback
- Hitta relevanta jobb med AI-sökning

**Koppling till EU-krav:** *"Bygga på kontinuerliga och verksamhetsnära former för lärande"*

---

### 3. AI-Etik & Ansvar

**Beskrivning:**
Modul om etiska överväganden vid AI-användning. Täcker bias, integritet, ansvar, och hur man använder AI ansvarsfullt i arbetslivet.

**Funktionalitet:**
- Scenariobaserade dilemman
- Diskussion om AI-bias och diskriminering
- Integritet och dataskydd
- Ansvarsfrågor: vem är ansvarig för AI-beslut?
- Regelverket: EU AI Act och svenska regler
- Praktiska riktlinjer för etisk AI-användning

**Ämnen:**
- Bias i rekrytering-AI
- Plagiat och AI-genererat innehåll
- Källkritik och faktakontroll av AI-svar
- Transparens: när bör man berätta att man använt AI?
- Arbetsgivarens vs arbetstagarens ansvar

**Koppling till EU-krav:** *"Stärka mänskligt omdöme, etik och ansvar i AI-användning"*

---

### 4. Framtidsjobb & AI-karta

**Beskrivning:**
Interaktiv visualisering av hur AI påverkar olika yrken och branscher. Visar vilka roller som förändras, vilka nya roller som uppstår, och vilka kompetenser som blir viktiga.

**Funktionalitet:**
- Välj yrke/bransch för att se AI-påverkan
- Risk-indikator: automatiseringsrisk per yrke
- Möjlighets-indikator: AI-förstärkning per yrke
- Nya roller som uppstår tack vare AI
- Kompetenser som blir mer/mindre viktiga
- Karriärvägar inom AI-påverkade branscher

**Datavisualisering:**
- Heatmap över branscher och AI-påverkan
- Tidslinje: när förväntas förändringen?
- Jämför ditt nuvarande yrke med framtidsscenarion
- Förslag på kompetenser att utveckla

**Branscher:**
- Vård och omsorg (AI för diagnostik, administration)
- Handel och service (chatbottar, lagerautomation)
- Industri och produktion (robotik, prediktivt underhåll)
- Administration och kontor (automation, AI-assistenter)
- Transport och logistik (autonoma fordon, ruttoptimering)
- IT och teknik (AI-utveckling, prompt engineering)
- Kreativa yrken (AI-genererat innehåll, samarbete människa-AI)

**Koppling till EU-krav:** *"Ta höjd för att AI skapar förändrade roller och arbetsprocesser"*

---

### 5. AI-Promptskola

**Beskrivning:**
Praktisk kurs i att kommunicera effektivt med AI-system. Lär ut tekniker för att skriva tydliga instruktioner (prompts) för att få bättre resultat från AI-verktyg.

**Funktionalitet:**
- Grunderna i promptning
- Strukturera dina förfrågningar
- Iterativt förbättra svar
- Rollspel och kontext
- Specifika tekniker: few-shot, chain-of-thought
- Praktiska övningar med poängsättning
- Promptbibliotek för jobbsökning

**Lektioner:**
1. Vad är en prompt?
2. Tydlighet och specificitet
3. Ge AI kontext och roll
4. Be om format och struktur
5. Iterera och förfina
6. Avancerade tekniker
7. Prompts för CV och personliga brev
8. Prompts för intervjuförberedelse

**Koppling till EU-krav:** *"Kompetensutvecklingsinsatser har genomförts i syfte att stärka deltagarnas kunskap och förmåga"*

---

### 6. AI-Beredskapsindex (Självskattning)

**Beskrivning:**
Omfattande självskattningsverktyg som mäter användarens AI-kompetens och beredskap för ett AI-drivet arbetsliv. Identifierar styrkor och utvecklingsområden.

**Funktionalitet:**
- Självskattning inom 6 kompetensområden
- Poängsatt resultat med visuell presentation
- Jämförelse mot arbetsmarknadens krav
- Personlig utvecklingsplan
- Koppling till relevanta kurser/moduler
- Uppföljning över tid (före/efter utbildning)

**Kompetensområden:**
1. **AI-förståelse** – Grundläggande kunskap om AI
2. **Teknisk förmåga** – Använda AI-verktyg praktiskt
3. **Kritiskt tänkande** – Utvärdera AI-resultat
4. **Etisk medvetenhet** – Förstå risker och ansvar
5. **Anpassningsförmåga** – Vilja att lära nytt
6. **Samarbete människa-AI** – Kombinera mänskliga och AI-styrkor

**Output:**
- AI-Beredskapspoäng (0-100)
- Styrkeanalys
- Utvecklingsplan med prioriterade områden
- Rekommenderade kurser/aktiviteter

**Koppling till EU-krav:** *Identifiera kompetensglapp*

---

### 7. AI-Rollsimulator

**Beskrivning:**
Interaktiv simulator som visar hur AI förändrar specifika arbetsroller. Användaren får prova på att arbeta med AI-stöd i olika scenarier och förstå hur deras framtida roll kan se ut.

**Funktionalitet:**
- Välj en yrkesroll
- Simulera arbetsuppgifter med och utan AI
- Jämför effektivitet och resultat
- Reflektera över vad som förändras
- Identifiera nya kompetenskrav
- Förstå samspelet människa-AI

**Scenarion:**
- **Administratör:** Automatisera rapporter, schemaläggning med AI
- **Kundtjänst:** Chatbot som första linje, mänsklig eskalering
- **Ekonomi:** AI för bokföring, analys och prognoser
- **HR:** CV-screening med AI, bias-medvetenhet
- **Marknadsföring:** AI-genererat innehåll, kampanjoptimering
- **Vård:** AI för triagering, dokumentation

**Koppling till EU-krav:** *"Ta höjd för att AI skapar förändrade roller och arbetsprocesser"*

---

### 8. AI-Säkerhet & Integritet

**Beskrivning:**
Utbildningsmodul om datasäkerhet och integritet i samband med AI-användning. Lär deltagare att skydda personlig information och använda AI säkert.

**Funktionalitet:**
- Vad händer med din data när du använder AI?
- Risker med att dela känslig information
- GDPR och AI
- Säker användning av AI-verktyg
- Identifiera AI-bedrägerier (deepfakes, phishing)
- Praktiska säkerhetstips

**Moduler:**
1. AI och dina personuppgifter
2. Vilken data samlar AI-tjänster in?
3. Säkra vs osäkra AI-verktyg
4. Hur du skyddar dig
5. AI-genererade bedrägerier
6. Dina rättigheter enligt GDPR

**Koppling till EU-krav:** *"Stärka mänskligt omdöme, etik och ansvar i AI-användning"*

### 9. Bransch-AI-Trender

**Beskrivning:**
Uppdaterad omvärldsbevakning av AI-utvecklingen inom olika branscher. Visar aktuella trender, nya verktyg, och vilka kompetenser som efterfrågas.

**Funktionalitet:**
- AI-nyheter per bransch
- Nya AI-verktyg och plattformar
- Jobbannons-analys: AI-relaterade krav
- Intervjuer med branschexperter
- Kurser och certifieringar per bransch
- Prenumerera på uppdateringar

**Branscher:**
- Vård och omsorg
- Handel och e-handel
- Bank och finans
- Industri och tillverkning
- IT och teknik
- Utbildning
- Offentlig sektor
- Kreativa branscher

**Koppling till EU-krav:** *"Organisationer har stärkt sin förmåga att förstå, använda och förhålla sig till AI"*

### 10. Konsulent AI-Coach

**Beskrivning:**
Specialdesignat stöd för arbetskonsulenter/handledare att vägleda deltagare i AI-frågor. Inkluderar utbildningsmaterial, samtalsguider, och verktyg för att bedöma deltagarnas AI-kompetens.

**Funktionalitet:**
- Konsulentutbildning i AI-grunderna
- Samtalsguider: prata om AI med deltagare
- Bedöm deltagarnas AI-beredskap
- Anpassa AI-utbildning efter deltagarens nivå
- Dashboard: följ deltagarnas AI-lärande
- Resurser för att svara på vanliga frågor

**Innehåll för konsulenter:**
- Crash course i AI (30 min)
- Hur AI påverkar olika målgrupper
- Vanliga frågor och svar om AI
- Samtalsmallar för AI-diskussioner
- Övningar att göra med deltagare
- Motivationsstrategier för AI-skeptiker

**Dashboard:**
- Deltagarnas AI-Beredskapsindex
- Genomförda AI-kurser
- Identifierade kompetensgap
- Rekommenderade nästa steg

**Koppling till EU-krav:** *"Beakta ledarskapets och organisationens roll i AI-omställningen"*

### Databasschema (Supabase)

**Nya tabeller:**

```sql
-- AI-lärandeprogress
CREATE TABLE ai_learning_progress (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES auth.users(id),
  module_id TEXT NOT NULL,
  module_type TEXT, -- 'guide' | 'ethics' | 'prompts' | 'security'
  status TEXT DEFAULT 'not_started', -- 'not_started' | 'in_progress' | 'completed'
  score INTEGER,
  time_spent_minutes INTEGER DEFAULT 0,
  completed_at TIMESTAMP,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

-- AI-Beredskapsindex
CREATE TABLE ai_readiness_assessments (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES auth.users(id),
  assessment_date DATE NOT NULL,
  ai_understanding INTEGER CHECK (ai_understanding BETWEEN 1 AND 10),
  technical_ability INTEGER CHECK (technical_ability BETWEEN 1 AND 10),
  critical_thinking INTEGER CHECK (critical_thinking BETWEEN 1 AND 10),
  ethical_awareness INTEGER CHECK (ethical_awareness BETWEEN 1 AND 10),
  adaptability INTEGER CHECK (adaptability BETWEEN 1 AND 10),
  human_ai_collaboration INTEGER CHECK (human_ai_collaboration BETWEEN 1 AND 10),
  total_score INTEGER,
  recommendations JSONB,
  created_at TIMESTAMP DEFAULT NOW()
);

-- AI-kursmoduler
CREATE TABLE ai_course_modules (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  module_type TEXT NOT NULL,
  title TEXT NOT NULL,
  description TEXT,
  duration_minutes INTEGER,
  difficulty_level TEXT, -- 'beginner' | 'intermediate' | 'advanced'
  order_index INTEGER,
  content JSONB,
  quiz_questions JSONB,
  created_at TIMESTAMP DEFAULT NOW()
);

-- Prompt-bibliotek
CREATE TABLE ai_prompt_library (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  category TEXT NOT NULL, -- 'cv' | 'cover_letter' | 'interview' | 'general'
  title TEXT NOT NULL,
  prompt_template TEXT NOT NULL,
  description TEXT,
  example_output TEXT,
  difficulty_level TEXT,
  rating DECIMAL(3,2),
  usage_count INTEGER DEFAULT 0,
  created_at TIMESTAMP DEFAULT NOW()
);

-- Konsulent AI-uppföljning
CREATE TABLE consultant_ai_tracking (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  consultant_id UUID REFERENCES auth.users(id),
  participant_id UUID REFERENCES auth.users(id),
  ai_readiness_score INTEGER,
  recommended_modules TEXT[],
  notes TEXT,
  last_reviewed TIMESTAMP,
  created_at TIMESTAMP DEFAULT NOW()
);
```
