# EU-projekt: Socialt innovativa insatser för ekonomiskt utsatta

**Diarienummer:** 26-010
**Utlysning:** Programområde E - Social Innovation
**Utlyst belopp:** 57 000 000 kr
**Max per projekt:** 4 750 000 kr
**Medfinansiering:** 5% (mycket låg!)
**Ansökningsperiod:** 2026-04-13 – 2026-09-03
**Projektperiod:** 2026-12-01 – 2029-06-30 (max 31 månader)

---

## Sammanfattning av EU-utlysningen

### Syfte
Utveckla socialt innovativa insatser för att möta behov hos ekonomiskt och socialt utsatta barn och ungdomar. Fokus på nyskapande lösningar, målgruppsinvolvering och nya samarbetsformer.

### Målgrupp
- **Barn 6-17 år:** I familjer med ekonomisk och social utsatthet
- **Ungdomar/unga vuxna 18-29 år:** I samma situation
- **Anställda:** Personal som arbetar med dessa grupper
- **Områden:** Särskilt utsatta kommuner/områden

### Specifika kommuner
Utlysningen riktar sig till:
- Borås, Botkyrka, Filipstad, Göteborg, Linköping
- Ljusnarsberg, Malmö, Stockholm, Södertälje
- Uppsala, Vingåker, Örebro

### Vad är Social Innovation?
Enligt ESF+ är social innovation verksamhet som är:

**Socialt i sina mål:**
- Öka människors välmående, inkludering och egenmakt
- Särskilt för dem i socialt eller ekonomiskt utsatt situation

**Socialt i sina medel:**
- Utvecklas genom kollektiva processer
- Nya sociala praktiker, relationer och strukturer
- Nytänkande samverkan mellan sektorer och organisationer

### Förväntade resultat

**Resultatmål:**
- Socialt innovativa medel och mål har använts
- Innovativa sätt att bedriva verksamhet för barn och unga utvecklats
- Nya arbetssätt i organisationen implementerats
- Fler barn och unga har högre närvaro och motivation till studier
- Ökat deltagande i fritidsaktiviteter
- Ökad känsla av sammanhang och inkludering

**Effektmål:**
- Innovativa och framgångsrika arbetssätt implementerade
- Organisationer använder nya metoder till gagn för målgruppen
- Ekonomisk och social utsatthet har minskat bland barn och unga

Projektet är högst relevant för portalen eftersom:

- Involverar målgruppen (deltagarcentrerat)
- Bygger på ny teknologi (AI)
- Skapar nya samarbetsformer (kommun + civilsamhälle + deltagare)

## Detaljerade funktionsbeskrivningar

### 1. Ungdomsportal 18-29 (Anpassad vy)

**Beskrivning:**
En specialdesignad version av portalen anpassad för unga vuxna 18-29 år. Förenklat språk, relevanta exempel, fokus på första jobbet och kombination med studier. Visuellt tilltalande design som känns modern och relevant.

**Funktionalitet:**
- Anpassat gränssnitt för yngre målgrupp
- Förenklat språk och tydliga instruktioner
- Relevanta jobbexempel (deltid, sommarjobb, extrajobb)
- Integrerat med studier (CSN, deltidsstudier)
- Gamification-element för motivation
- Mobil-först-design
- Tillgänglighetsanpassad (WCAG 2.1 AA)

**Innehåll:**
- "Ditt första CV" - guide från scratch
- "Så funkar arbetsmarknaden" - grundläggande info
- "Rättigheter som anställd" - vad du ska veta
- "Kombinera jobb och studier" - tips och planering
- "Hantera pengar" - grundläggande ekonomi

**Koppling till EU-krav:** *Socialt innovativa sätt att bedriva verksamhet för barn och unga*

---

### 2. Första Jobbet-guide

**Beskrivning:**
Komplett guide för den som aldrig haft ett jobb. Steg-för-steg från "jag har inget att skriva på CV:t" till första anställningen. Normaliserar att börja från noll.

**Funktionalitet:**
- Steg-för-steg-process med tydliga milstolpar
- AI-hjälp att identifiera dolda kompetenser
- Skapa CV utan arbetslivserfarenhet
- Hitta rätt typ av jobb (deltid, praktik, feriejobb)
- Förberedelse för första intervjun
- Vad händer efter du fått jobbet?

**Steg i guiden:**
1. Vem är du? (intressen, styrkor, drömmar)
2. Vad kan du? (skola, fritid, volontärarbete)
3. Skapa ditt första CV
4. Hitta rätt jobb för dig
5. Skicka din första ansökan
6. Förbered dig för intervju
7. Du fick jobbet - vad nu?

**Koppling till EU-krav:** *Stegförflyttning mot arbetsmarknaden*

---

### 3. Studier & Jobb-planering

**Beskrivning:**
Verktyg för att planera kombination av studier och arbete. Hjälper unga att se möjligheter att jobba parallellt med studier, eller planera en väg via studier till drömjobbet.

**Funktionalitet:**
- Veckoschema-planerare (studier + jobb + fritid)
- CSN-kalkylator (hur mycket får jag jobba?)
- Jobbfilter: "Passar med studier"
- Karriärvägar: "Studera till ditt drömjobb"
- Information om YH, Komvux, folkhögskola
- Praktik och traineeprogram

**Integreringar:**
- Skolverket - utbildningsinfo
- CSN - bidragsregler
- Arbetsförmedlingen - yrkesprognoser
- Lokala utbildningsanordnare

**Koppling till EU-krav:** *Högre närvaro och motivation till studier och vidareutbildning*

---

### 4. Fritid-till-Kompetens

**Beskrivning:**
Innovativt verktyg som hjälper unga att omvandla sina fritidsaktiviteter till CV-meriter och kompetenser. Visar att gaming, sport, föreningsliv och hobbyer faktiskt utvecklar värdefulla färdigheter.

**Funktionalitet:**
- Välj dina fritidsaktiviteter
- AI identifierar utvecklade kompetenser
- Generera formuleringar för CV
- Koppla till relevanta yrken
- Skapa "Kompetensbevis" för fritidsaktiviteter

**Exempel på kopplingar:**
- **Gaming:** Problemlösning, teamwork, strategi, engelska
- **Sport:** Disciplin, samarbete, stresshantering, målfokus
- **Föreningsliv:** Ledarskap, organisation, kommunikation
- **Musik:** Kreativitet, övning/tålamod, framträdande
- **Sociala medier:** Digital kompetens, kommunikation, marknadsföring
- **Barnpassning:** Ansvar, flexibilitet, pedagogik

**Koppling till EU-krav:** *Högre deltagande i fritidsaktiviteter* och *synliggöra informella kompetenser*

---

### 5. Min Framtidskarta (Visuell karriärplanering)

**Beskrivning:**
Visuell, interaktiv karta som visar vägen från nuläge till drömjobb. Skapar sammanhang och motivation genom att visa att det finns en väg framåt, även om den är lång.

**Funktionalitet:**
- Definiera ditt mål (drömjobb eller livsstil)
- Kartlägg nuläge (utbildning, erfarenhet, nätverk)
- AI föreslår möjliga vägar
- Visualisera stegen på kartan
- Sätt delmål och milstolpar
- Fira framsteg

**Visualisering:**
- Interaktiv tidslinje/karta
- Tydliga steg och mellanmål
- "Du är här"-markör
- Alternativa vägar/spår
- Inspirationsberättelser från andra

**Psykologisk design:**
- Skapa känsla av kontroll
- Visa att förändring är möjlig
- Normalisera att det tar tid
- Fira små framsteg

**Koppling till EU-krav:** *Ökad känsla av sammanhang och inkludering*

---

### 6. Familjeekonomiguide

**Beskrivning:**
Resurser och verktyg för att förbättra familjens ekonomiska situation. Riktar sig till föräldrar i ekonomisk utsatthet som vill komma in på arbetsmarknaden för att förbättra sin och barnens situation.

**Funktionalitet:**
- Budgetverktyg (inkomster vs utgifter)
- Information om bidrag och stöd (Försäkringskassan, kommun)
- Vägar till arbete för föräldrar (deltid, flexibla jobb)
- Barnomsorg och jobbsökning
- Skuldrådgivning-resurser
- Kontakt med kommun/socialtjänst

**Innehåll:**
- "Jobbsöka som förälder" - guide
- "Dessa bidrag har du rätt till"
- "Så fungerar A-kassa"
- "Skulder och kronofogden - vad gäller?"
- "Kommunens stöd för ekonomiskt utsatta"

**Koppling till EU-krav:** *Ekonomisk och social utsatthet har minskat*

---

### 7. Nätverksbyggare

**Beskrivning:**
Verktyg som hjälper unga utan professionellt nätverk att börja bygga kontakter. Många jobb hittas via nätverk - denna funktion hjälper dem som inte har "rätt kontakter".

**Funktionalitet:**
- Kartlägg ditt befintliga nätverk
- Tips på hur du utökar nätverket
- Bransch-specifika nätverksevent
- Digital kontaktbok med påminnelser
- Övningar i att ta kontakt
- Informationsintervjuer - hur gör man?

**Strategier:**
- LinkedIn för nybörjare
- Branschträffar och mässor
- Volontärarbete som nätverksbyggande
- Praktik som dörröppnare
- Informationsintervjuer
- Sociala medier för jobbsökning

**Koppling till EU-krav:** *Skapar nya sociala relationer*

---

### 8. Mentorsmatchning

**Beskrivning:**
Plattform för att koppla ihop unga jobbsökande med erfarna yrkesmentorer. Skapar de relationer och det stöd som målgruppen ofta saknar.

**Funktionalitet:**
- Matchningsalgoritm (intresse, bransch, bakgrund)
- Profiler för mentorer och adepter
- Strukturerade mentorsträffar (guider)
- Meddelandefunktion
- Mötesbokningssystem
- Uppföljning och feedback

**Mentortyper:**
- Yrkesmentorer (specifik bransch)
- Livserfarenhetsmentorer (liknande bakgrund)
- Studie-/karriärcoacher
- Språkmentorer (svenska/engelska)
- Entreprenörsmentorer

**Samarbetspartners:**
- Kommuner
- Företag (CSR-program)
- Föreningar (Rotary, Lions, etc.)
- Branschorganisationer
- Civilsamhälle (Mentor Sverige, etc.)

**Koppling till EU-krav:** *Nytänkande samverkan mellan olika sektorer*

---

### 9. Inkluderingsindex (Känsla av sammanhang)

**Beskrivning:**
Verktyg för självskattning av känsla av inkludering, sammanhang och hopp om framtiden. Baserat på forskning om KASAM (Känsla av sammanhang). Används för uppföljning och att identifiera behov av extra stöd.

**Funktionalitet:**
- Regelbunden självskattning (varannan vecka)
- Mätning av tre dimensioner: begriplighet, hanterbarhet, meningsfullhet
- Visuell presentation av utveckling över tid
- AI-förslag på aktiviteter för att öka inkludering
- Flaggning vid låga värden (för konsulent)
- Anonymiserad aggregering för utvärdering

**Dimensioner (KASAM):**
1. **Begriplighet:** Förstår jag vad som händer i mitt liv?
2. **Hanterbarhet:** Har jag resurser att hantera utmaningar?
3. **Meningsfullhet:** Finns det mening med det jag gör?

**Integration:**
- Wellness-modulen (Hälsa)
- Dagboken
- Konsulent-dashboard
- Projektrapportering till ESF

**Koppling till EU-krav:** *Ökad känsla av sammanhang och inkludering* - direkt mätbart

---

### 10. Kommun-dashboard (Uppföljning för projektaktörer)

**Beskrivning:**
Dashboard för kommunala projektledare och handledare att följa deltagarnas framsteg, aggregera data för rapportering till ESF, och identifiera mönster och behov.

**Funktionalitet:**
- Översikt över alla deltagare (anonymiserat)
- Framstegsindikator per deltagare
- Aggregerad statistik för rapportering
- Identifiera deltagare som behöver extra stöd
- Export till ESF:s rapporteringsformat
- Jämförelse mellan kommuner (benchmarking)

**Rapporterings-KPI:er:**
- Antal aktiva deltagare
- Genomsnittlig aktivitetsnivå
- Inkluderingsindex (genomsnitt, förändring)
- Antal som nått delmål
- Antal i arbete/praktik/studier
- Fritidsaktiviteter (ökning)

**Funktioner för handledare:**
- Se individuella framsteg
- Skicka uppmuntrande meddelanden
- Boka möten med deltagare
- Dokumentera insatser
- Dela resurser med deltagare

**Koppling till EU-krav:** *Rapportering av deltagare till SCB* och *uppföljningsbara projektmål*

-- Ungdomsprofiler
CREATE TABLE youth_profiles (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES auth.users(id),
  age_group TEXT CHECK (age_group IN ('18-24', '25-29')),
  municipality TEXT,
  education_status TEXT, -- 'studying' | 'completed' | 'dropout' | 'planning'
  employment_status TEXT, -- 'employed' | 'unemployed' | 'first_job_seeker'
  has_work_experience BOOLEAN DEFAULT false,
  interests TEXT[],
  barriers TEXT[], -- språk, funktionsnedsättning, etc.
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()

CREATE TABLE leisure_activities (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),

  activity_type TEXT NOT NULL,
  activity_name TEXT NOT NULL,
  duration_years INTEGER,
  skills_identified TEXT[],
  cv_description TEXT,
  created_at TIMESTAMP DEFAULT NOW()
);

-- Inkluderingsindex (KASAM)
CREATE TABLE inclusion_assessments (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES auth.users(id),
  assessment_date DATE NOT NULL,
  comprehensibility INTEGER CHECK (comprehensibility BETWEEN 1 AND 7),
  manageability INTEGER CHECK (manageability BETWEEN 1 AND 7),
  meaningfulness INTEGER CHECK (meaningfulness BETWEEN 1 AND 7),
  total_score INTEGER,
  notes TEXT,
  created_at TIMESTAMP DEFAULT NOW()
);

-- Framtidskarta-mål
CREATE TABLE future_map_goals (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES auth.users(id),
  goal_type TEXT, -- 'dream_job' | 'milestone' | 'step'
  title TEXT NOT NULL,
  description TEXT,
  target_date DATE,
  status TEXT DEFAULT 'not_started',
  parent_goal_id UUID REFERENCES future_map_goals(id),
  order_index INTEGER,
  created_at TIMESTAMP DEFAULT NOW()
);

-- Mentorsrelationer
CREATE TABLE mentorship_relationships (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  mentor_id UUID REFERENCES auth.users(id),
  mentee_id UUID REFERENCES auth.users(id),
  match_date TIMESTAMP DEFAULT NOW(),
  status TEXT DEFAULT 'active', -- 'active' | 'paused' | 'completed'
  match_reason TEXT,
  meetings_count INTEGER DEFAULT 0,
  next_meeting TIMESTAMP,
  notes TEXT
);

-- Kommun-projektdeltagare
CREATE TABLE municipality_participants (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES auth.users(id),
  municipality TEXT NOT NULL,
  project_id TEXT, -- ESF-projekt-ID
  enrolled_date DATE NOT NULL,
  status TEXT DEFAULT 'active',
  consultant_id UUID REFERENCES auth.users(id),
  progress_score INTEGER,
  last_activity TIMESTAMP,
  created_at TIMESTAMP DEFAULT NOW()
);
